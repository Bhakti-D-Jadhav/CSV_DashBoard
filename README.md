# 📊 CSV Dashboard Generator

A beginner-friendly data analytics application built with **Python**, **Streamlit**, **Pandas**, **Plotly**, and **ReportLab**. This application allows users to upload a CSV file, explore the dataset, visualize data through interactive charts, and download a professional PDF report.

---

## 🚀 Features

* 📂 Upload any CSV file
* 📄 View dataset preview
* 📊 Display dataset information

  * Total Rows
  * Total Columns
  * Missing Values
* 📈 Generate summary statistics
* 📉 Create interactive visualizations:

  * Bar Chart
  * Line Chart
  * Pie Chart
* 📄 Generate and download a PDF dashboard report
* 🎨 Simple and user-friendly interface

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/your-username/CSV_Dashboard.git
cd CSV_Dashboard
```

### Create a Virtual Environment

```bash
python -m venv .venv
```

### Activate the Virtual Environment

**Windows**

```bash
.venv\Scripts\activate
```

**Linux/Mac**

```bash
source .venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

Open your browser and visit:

```text
http://localhost:8501
```
